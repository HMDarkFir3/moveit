## How to run the app 

Inside a folder, run:
```bash
$ git clone https://github.com/HMDarkFir3/NLW-04_move.it.git
```
At the root of the project, run:
```bash
npm install  
or 
yarn install
```
After installing the project dependencies, run:
```bash
npm run dev
or
yarn dev
```
